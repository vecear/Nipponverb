# 日文單字教學內容生成指南

> 此技能指導 AI 依據單字列表，編寫完整的單字教學內容，包含中文意思、解說、小提醒及例句

---

## 快速檢查清單

執行前後請確認以下項目：

- [ ] 每個單字包含三大部分：中文意思、解說、小提醒
- [ ] 每個單字有 3 句例句（2 句單句 + 1 句對話）
- [ ] 所有漢字都標註 `{平假名}`
- [ ] 例句附有中文翻譯
- [ ] 執行 `npx tsc --noEmit` 無錯誤

---

## 1. 核心原則

| 原則 | 說明 |
|------|------|
| 逐詞手寫 | 禁止腳本批量生成，確保教學內容正確 |
| 台灣繁體中文 | 使用自然的台灣口語 |
| 實用導向 | 例句選擇日常生活常用場景 |
| 易錯點提醒 | 特別標註台灣學習者常犯的錯誤 |

### 1.1 LLM 模式（重要）

**當使用者明確表示「使用 LLM」時**，一律：

| 規則 | 說明 |
|------|------|
| ❌ 禁止程式 | 不使用任何程式語言撰寫腳本或自動化流程 |
| ✅ 直接操作 | 由 LLM 直接讀取、理解、檢視、修正檔案內容 |
| ✅ 逐詞處理 | LLM 逐一編寫每個單字的教學內容 |
| ✅ 使用編輯工具 | 使用 Read / Edit / Write 等工具直接操作檔案 |

## 2. 單字資料來源

**單字列表路徑**：`src/data/vocabulary.ts`

單字資料結構：
```typescript
interface VocabularyData {
    word: string       // 漢字或假名
    reading: string    // 平假名讀音
    meaning: string    // 英文意思
    meaning_zh: string // 中文意思
    level: 'N5' | 'N4' | 'N3' | 'N2' | 'N1'
}
```

---

## 3. 教學內容結構

每個單字的教學內容包含以下三大部分：

### 3.1 第一部分：中文意思

簡潔明瞭的中文解釋，包含：
- 主要意思
- 詞性標註
- 常見的多重意思（如有）

### 3.2 第二部分：解說

說明這個單字：
- 用在什麼場合
- 什麼時候使用
- 常見的搭配用法
- 語感說明（正式/非正式/口語等）

### 3.3 第三部分：小提醒

補充台灣學習者需要注意的內容：
- 常搞混的相似詞
- 發音容易錯的地方
- 漢字容易寫錯的部分
- 文化差異說明
- 敬語使用注意事項

---

## 4. 例句規範

每個單字需要 **3 句例句**：

| 例句 | 類型 | 說明 |
|------|------|------|
| 例句 1 | 簡單單句 | 最基本的使用方式，初學者也能理解 |
| 例句 2 | 實用單句 | 稍微複雜，展示不同用法或搭配 |
| 例句 3 | 兩人對話 | 模擬真實對話場景，展示實際運用 |

### 4.1 例句格式規範

**單句格式**：
```
日文句子（漢字標註{假名}）
中文翻譯
```

**對話格式**：
```
A：「日文句子」
B：「日文句子」
A：「中文翻譯」
B：「中文翻譯」
```

### 4.2 漢字標註

所有日文漢字都必須標註平假名：

| 規則 | 範例 |
|------|------|
| ✅ 正確 | `明{あか}るい部屋{へや}` |
| ❌ 錯誤 | `明るい部屋` |

---

## 5. 資料結構

**寫入路徑**：`src/data/vocabulary/details/n{level}_vocab_details.ts`

```typescript
export interface VocabDetail {
    id: string                    // 單字 ID
    word: string                  // 漢字或假名
    reading: string               // 平假名讀音
    meaning_zh: string            // 中文意思
    partOfSpeech: string          // 詞性
    explanation: string           // 解說（用在什麼地方）
    reminder: string              // 小提醒（常搞混的點）
    examples: VocabExample[]      // 例句（3句）
    level: 'N5' | 'N4' | 'N3' | 'N2' | 'N1'
}

export interface VocabExample {
    japanese: string              // 日文例句（含假名標註）
    chinese: string               // 中文翻譯
    type: 'simple' | 'practical' | 'dialogue'  // 例句類型
}
```

2. **性格開朗**
   用來形容人的個性樂觀、活潑。

  「明{あき}らか」是「明顯的、清楚的」

### 範例 2：食べる（N5 動詞）

```typescript
{
    id: 'n5_vocab_taberu',
    word: '食べる',
    reading: 'たべる',
    meaning_zh: '吃',
    partOfSpeech: '一段動詞（他動詞）',
    explanation: `【用法說明】
「食{た}べる」是表示「吃」的基本動詞。

**基本句型**
・〇〇を食{た}べる（吃〇〇）
  例：ご飯{はん}を食{た}べる（吃飯）

**常見搭配**
・朝{あさ}ご飯{はん}を食{た}べる（吃早餐）
・昼{ひる}ご飯{はん}を食{た}べる（吃午餐）
・晩{ばん}ご飯{はん}を食{た}べる（吃晚餐）
・おやつを食{た}べる（吃點心）

【語感】
日常用語，正式和非正式場合都可使用。
更禮貌的說法是「いただく」或「召{め}し上{あ}がる」（敬語）。`,

    reminder: `【小提醒】

⚠️ **敬語變化**
・謙讓語：いただく
  例：お寿司{すし}をいただきます。（我吃壽司。）
・尊敬語：召{め}し上{あ}がる
  例：どうぞ召{め}し上{あ}がってください。（請享用。）

⚠️ **動詞變化（一段動詞）**
・ます形：食{た}べます
・て形：食{た}べて
・ない形：食{た}べない
・た形：食{た}べた
・可能形：食{た}べられる

⚠️ **「食べる」vs「飲む」**
・固體食物 → 食{た}べる
・液體飲料 → 飲{の}む
・藥、湯品 → 飲{の}む（日文中藥和湯是「喝」的）

⚠️ **常見錯誤**
✕ パンを飲{の}む（麵包要用「食べる」）
○ パンを食{た}べる`,

    examples: [
        {
            japanese: '毎朝{まいあさ}、パンを食{た}べます。',
            chinese: '每天早上吃麵包。',
            type: 'simple'
        },
        {
            japanese: '日本{にほん}に行{い}ったら、お寿司{すし}を食{た}べたいです。',
            chinese: '去日本的話，想吃壽司。',
            type: 'practical'
        },
        {
            japanese: 'A：「お昼{ひる}、何{なに}食{た}べる？」\nB：「うーん、ラーメンはどう？」\nA：「いいね！行{い}こう！」',
            chinese: 'A：「中午吃什麼？」\nB：「嗯...拉麵怎麼樣？」\nA：「好啊！走吧！」',
            type: 'dialogue'
        }
    ],
    level: 'N5'
}
```

## 7. 各詞性的解說重點

### 7.1 動詞

| 項目 | 說明 |
|------|------|
| 自他動詞 | 說明是自動詞還是他動詞，列出對應的另一個 |
| 動詞分類 | 一段/五段/不規則動詞 |
| 常見句型 | 〇〇を～、〇〇に～ 等 |
| 變化形 | ます形、て形、ない形、た形、可能形 |
| 敬語 | 謙讓語、尊敬語（如適用） |

## 9. 工作流程

1. **選擇單字**：從 `src/data/vocabulary.ts` 選擇要編寫的單字
2. **查詢資料**：確認單字的詞性、意思、用法
3. **編寫三大部分**：
   - 中文意思（簡潔明瞭）
   - 解說（用法、場合、搭配）
   - 小提醒（易錯點、相似詞）
4. **編寫例句**：
   - 例句 1：簡單單句
   - 例句 2：實用單句
   - 例句 3：兩人對話
5. **標註假名**：確認所有漢字都標註 `{假名}`
6. **檢查格式**：確認符合規範
7. **執行驗證**：`npx tsc --noEmit`

## 10. 練習題目（教學結尾）

每個單字教學完成後，必須在結尾附上 **3 題練習題目**，讓學習者立即複習。

### 10.1 題目規範

練習題目須遵循 `VOCAB_SKILL.md` 的出題規範：

| 規則 | 說明 |
|------|------|
| 題數 | 3 題 |
| 選項數 | 每題 4 個選項 |
| 正確答案位置 | 隨機分布於 0-3（不可固定在同一位置） |
| 選項解釋 | 每個選項都有 `reason` 欄位說明對錯原因 |
| 詳解 | 每題都有 `correctRule` 提供完整解析 |

### 10.2 題型選擇

從以下題型中選擇 3 種不同的題型：

| 題型 | 說明 | 適用級別 |
|------|------|----------|
| 漢字読み | 選出劃線漢字的正確平假名 | N5-N1 |
| 表記 | 根據平假名選出正確漢字 | N5-N1 |
| 文脈規定 | 根據句意選出最符合語境的單字 | N5-N1 |
| 類義語 | 找出與劃線單字意義最接近的選項 | N5-N1 |
| 用法 | 單字在句中是否使用正確 | N3-N1 |
| 語形成 | 前綴詞/後綴詞的組合 | N2-N1 |

**建議組合**：

- N5/N4：漢字読み + 表記 + 文脈規定
- N3：漢字読み + 文脈規定 + 用法
- N2/N1：文脈規定 + 用法 + 語形成

### 10.3 資料結構

在 `VocabDetail` 中新增 `practiceQuestions` 欄位：

```typescript
export interface VocabDetail {
    // ... 原有欄位 ...
    practiceQuestions: PracticeQuestion[]  // 練習題目（3題）
}

export interface PracticeQuestion {
    id: string
    prob: string                    // 日文題目（含假名標註）
    prob_zh: string                 // 中文翻譯（用 *答案* 標記）
    options: {
        text: string                // 選項文字
        reason: string              // 對錯原因說明
    }[]
    correctIndex: number            // 正確答案位置 (0-3)
    tag: string                     // 題型標籤
    correctRule: string             // 詳解
}
```

### 10.4 完整範例

以「明るい」為例，教學結尾的練習題目：

```typescript
practiceQuestions: [
    // 題目 1：漢字読み（正確答案在位置 2）
    {
        id: 'n5_vocab_practice_akarui_1',
        prob: 'この部屋{へや}は<u>明るい</u>ですね。',
        prob_zh: '這個房間很 *明亮* 呢。',
        options: [
            { text: 'あかるうい', reason: '錯誤：多了長音，正確讀音沒有長音。' },
            { text: 'あがるい', reason: '錯誤：清濁音錯誤，「か」不是「が」。' },
            { text: 'あかるい', reason: '正確！「明るい」讀作「あかるい」。' },
            { text: 'あっかるい', reason: '錯誤：多了促音，正確讀音沒有促音。' }
        ],
        correctIndex: 2,
        tag: 'vocab_kanji_yomi',
        correctRule: '【考點】漢字讀音\n【解析】「明{あか}るい」是 i 形容詞，意思是「明亮的」。\n【易錯點】\n・○ あかるい\n・✕ あがるい（清濁音錯誤）\n・✕ あかるうい（長音錯誤）'
    },
    // 題目 2：表記（正確答案在位置 0）
    {
        id: 'n5_vocab_practice_akarui_2',
        prob: 'へやが<u>あかるい</u>です。',
        prob_zh: '房間很 *明亮*。',
        options: [
            { text: '明るい', reason: '正確！「あかるい」的漢字是「明るい」。' },
            { text: '開るい', reason: '錯誤：「開」讀作「あ（ける）」，意思是打開。' },
            { text: '空るい', reason: '錯誤：「空」讀作「あ（く）」，意思是變空。' },
            { text: '赤るい', reason: '錯誤：「赤」讀作「あか」，無「あかるい」讀法。' }
        ],
        correctIndex: 0,
        tag: 'vocab_hyouki',
        correctRule: '【考點】漢字書寫\n【解析】「明{あか}るい」的漢字為「明」。\n【同音字比較】\n・明{あか}るい：明亮\n・赤{あか}い：紅色'
    },
    // 題目 3：文脈規定（正確答案在位置 3）
    {
        id: 'n5_vocab_practice_akarui_3',
        prob: '朝{あさ}になって、外{そと}が______なりました。',
        prob_zh: '到了早上，外面變 *亮* 了。',
        options: [
            { text: '暗{くら}く', reason: '錯誤：「暗い」意思是暗，與早上語境相反。' },
            { text: '広{ひろ}く', reason: '錯誤：「広い」意思是寬廣，不形容光線。' },
            { text: '長{なが}く', reason: '錯誤：「長い」意思是長，不形容光線。' },
            { text: '明{あか}るく', reason: '正確！「明るい」→「明るく」＋「なる」表示變亮。' }
        ],
        correctIndex: 3,
        tag: 'vocab_bunmyaku',
        correctRule: '【考點】文脈語境\n【解析】早上天亮，選「明{あか}るく」。\n【文法】i形容詞 + く + なる = 變得～'
    }
]
```

## 11. 注意事項

1. **漢字標註**：所有日文漢字都要加 `{假名}` 標註
2. **對話格式**：對話用 `A：「」\nB：「」` 格式
3. **中文用詞**：使用台灣繁體中文和自然口語
4. **例句難度**：
   - N5/N4：使用該級別的文法和詞彙
   - N3 以上：可使用稍複雜的句型
5. **實用性**：例句選擇日常生活常見場景
6. **解說深度**：
   - N5/N4：著重基本用法
   - N3 以上：可補充進階用法和語感差異
7. **練習題目**：
   - 每個單字教學結尾附上 3 題練習題
   - 正確答案位置隨機分布（0-3）
   - 每個選項都要有對錯原因說明
