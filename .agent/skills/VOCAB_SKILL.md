# 日文單字題目生成指南

> 此技能指導 AI 依據 JLPT 正式考試的「言語知識（文字・語彙）」格式，手動撰寫日文單字題目

---

## 快速檢查清單

執行前後請確認以下項目：

- [ ] 每個單字產生 4 種題型，每種題型 2 題（共 8 題）
- [ ] 題目 1 為簡單單句，題目 2 為兩人對話
- [ ] `prob_zh` 使用 `*答案*` 格式
- [ ] `correctIndex` 隨機分布於 0-3
- [ ] 錯誤選項必須有明確錯誤原因
- [ ] 執行 `npx tsc --noEmit` 無錯誤
- [ ] N5/N4 只出前 4 種題型，N3 以上可出全部 6 種題型

## 1. 核心原則

| 原則 | 說明 |
|------|------|
| 逐題手寫 | 禁止腳本批量生成，確保語境與語意正確 |
| 台灣繁體中文 | 翻譯使用自然的台灣口語 |
| 雙語詳解 | 所有解釋附完整中文翻譯 |
| 仿 JLPT 格式 | 題目格式須符合 JLPT 正式考試樣式 |

### 1.1 LLM 模式（重要）

**當使用者明確表示「使用 LLM」時**，一律：

| 規則 | 說明 |
|------|------|
| ❌ 禁止程式 | 不使用任何程式語言撰寫腳本或自動化流程 |
| ✅ 直接操作 | 由 LLM 直接讀取、理解、檢視、修正檔案內容 |
| ✅ 逐題處理 | LLM 逐一閱讀每題內容，人工判斷並修正 |
| ✅ 使用編輯工具 | 使用 Read / Edit / Write 等工具直接操作檔案 |

## 2. 單字資料來源

**單字列表路徑**：`src/data/vocabulary.ts`

單字資料結構：
```typescript
interface VocabularyData {
    word: string       // 漢字或假名
    reading: string    // 平假名讀音
    meaning: string    // 英文意思
    meaning_zh: string // 中文意思
    level: 'N5' | 'N4' | 'N3' | 'N2' | 'N1'
}
```

**範例**：
```typescript
{ word: '明るい', reading: 'あかるい', meaning: 'bright', meaning_zh: '明亮的', level: 'N5' }
```

## 3. JLPT 單字考題六大題型

### 3.1 題型總覽

| 題型 | 日文名稱 | 考察重點 | 適用級別 |
|------|----------|----------|----------|
| 1 | 漢字読み | 選出劃線漢字的正確平假名 | N5-N1 |
| 2 | 表記 | 根據平假名選出正確漢字 | N5-N1 |
| 3 | 文脈規定 | 根據句意選出最符合語境的單字 | N5-N1 |
| 4 | 語形成 | 前綴詞/後綴詞的組合 | N2-N1 |
| 5 | 類義語 | 找出與劃線單字意義最接近的選項 | N5-N1 |
| 6 | 用法 | 單字在句中是否使用正確（語法搭配） | N5-N1 |

### 3.2 各題型詳細說明

#### 題型 1：漢字読み（漢字讀音）

**考察重點**：是否能正確選出劃線漢字的平假名讀音

**常見陷阱**：
- 長音/短音混淆（びよういん vs びょういん）
- 清音/濁音混淆（はし vs ばし）
- 促音的有無（いっぱい vs いぱい）
- 特殊讀音（今日{きょう}、明日{あした}）

**題目格式**：
```typescript
{
    id: 'n5_vocab_kanji_yomi_akarui_1',
    prob: 'この部屋{へや}はとても<u>明るい</u>ですね。',
    prob_zh: '這個房間非常 *明亮* 呢。',
    options: [
        { text: 'あかるい', reason: '正確！「明るい」讀作「あかるい」。' },
        { text: 'あかるうい', reason: '錯誤：多了長音，正確讀音沒有長音。' },
        { text: 'あがるい', reason: '錯誤：清濁音錯誤，「か」不是「が」。' },
        { text: 'あっかるい', reason: '錯誤：多了促音，正確讀音沒有促音。' }
    ],
    correctIndex: 0,
    level: 'N5',
    tag: 'vocab_kanji_yomi',
    correctRule: '【考點】漢字讀音\n【解析】「明{あか}るい」是 i 形容詞，意思是「明亮的」。\n【易錯點】\n・○ あかるい\n・✕ あがるい（清濁音錯誤）\n・✕ あかるうい（長音錯誤）'
}
```

#### 題型 2：表記（漢字書寫）

**考察重點**：根據平假名選出正確的漢字

**常見陷阱**：
- 同音異義字（橋/箸/端 都讀 はし）
- 形似字（持/待、買/貝）

**題目格式**：
```typescript
{
    id: 'n5_vocab_hyouki_akarui_1',
    prob: 'へやが<u>あかるい</u>です。',
    prob_zh: '房間很 *明亮*。',
    options: [
        { text: '明るい', reason: '正確！「あかるい」的漢字是「明るい」。' },
        { text: '開るい', reason: '錯誤：「開」讀作「あ（ける）」或「ひら（く）」，意思是打開。' },
        { text: '空るい', reason: '錯誤：「空」讀作「あ（く）」或「そら」，意思是空/天空。' },
        { text: '赤るい', reason: '錯誤：「赤」讀作「あか」，意思是紅色，無「あかるい」的讀音。' }
    ],
    correctIndex: 0,
    level: 'N5',
    tag: 'vocab_hyouki',
    correctRule: '【考點】漢字書寫\n【解析】「明{あか}るい」的漢字為「明」，表示光亮、明亮。\n【同音異義字比較】\n・明{あか}るい：明亮的\n・開{あ}ける：打開\n・空{あ}く：變空\n・赤{あか}い：紅色的'
}
```

#### 題型 3：文脈規定（文脈定義）

**考察重點**：根據句意，選出最符合語境的單字

**常見陷阱**：
- 副詞的細微差別（だんだん vs どんどん vs ますます）
- 擬聲擬態語的區別
- 外來語（カタカナ）的正確使用

**題目格式**：
```typescript
{
    id: 'n5_vocab_bunmyaku_akarui_1',
    prob: '朝{あさ}になって、外{そと}が______なりました。',
    prob_zh: '到了早上，外面變 *亮* 了。',
    options: [
        { text: '明{あか}るく', reason: '正確！「明るい」的く形，接「なる」表示變得明亮。' },
        { text: '暗{くら}く', reason: '錯誤：「暗い」意思是暗，與早上變亮的語境相反。' },
        { text: '広{ひろ}く', reason: '錯誤：「広い」意思是寬廣，不適用於形容光線。' },
        { text: '長{なが}く', reason: '錯誤：「長い」意思是長，不適用於形容光線。' }
    ],
    correctIndex: 0,
    level: 'N5',
    tag: 'vocab_bunmyaku',
    correctRule: '【考點】文脈語境判斷\n【解析】早上天亮，要選擇「明{あか}るく」（變亮）。\n【相關詞比較】\n・明{あか}るい：明亮的\n・暗{くら}い：昏暗的\n【文法補充】i形容詞 + く + なる = 變得～'
}
```

#### 題型 4：語形成（詞彙組成）- N2 以上限定

**考察重點**：前綴詞（接頭辭）或後綴詞（接尾辭）的組合

**常見前綴詞**：
- 不～（不可能）、無～（無意味）、未～（未完成）
- 大～（大成功）、小～（小規模）
- 再～（再確認）、逆～（逆輸入）

**常見後綴詞**：
- ～性（可能性）、～的（積極的）、～化（国際化）
- ～者（研究者）、～員（会社員）、～家（専門家）

**題目格式**：
```typescript
{
    id: 'n2_vocab_gokeisei_kanousei_1',
    prob: 'この計画{けいかく}が成功{せいこう}する（　　）は低{ひく}い。',
    prob_zh: '這個計畫成功的 *可能性* 很低。',
    options: [
        { text: '可能性{かのうせい}', reason: '正確！「性」接在名詞後表示～的性質/可能。' },
        { text: '可能的{かのうてき}', reason: '錯誤：「的」用於形容詞化，如「積極的」，此處需要名詞。' },
        { text: '可能化{かのうか}', reason: '錯誤：「化」表示變化過程，如「国際化」，此處語意不通。' },
        { text: '可能感{かのうかん}', reason: '錯誤：「感」表示感覺，如「違和感」，此處語意不通。' }
    ],
    correctIndex: 0,
    level: 'N2',
    tag: 'vocab_gokeisei',
    correctRule: '【考點】接尾辭「～性」的用法\n【解析】「性{せい}」接在名詞後，表示「～的性質」或「～的可能」。\n【常見「～性」詞彙】\n・可能性{かのうせい}：可能性\n・重要性{じゅうようせい}：重要性\n・必要性{ひつようせい}：必要性\n・安全性{あんぜんせい}：安全性'
}
```

#### 題型 5：類義語（近義詞替換）

**考察重點**：找出與題目劃線單字意義最接近的選項

**題目格式**：
```typescript
{
    id: 'n5_vocab_ruigigo_akarui_1',
    prob: 'この部屋{へや}は<u>明るい</u>です。',
    prob_zh: '這個房間很 *明亮*。',
    question_type: '劃線部分的單字與下列何者意思最接近？',
    options: [
        { text: '光{ひかり}がたくさんある', reason: '正確！「明るい」＝有很多光線，意思最接近。' },
        { text: '暗{くら}い', reason: '錯誤：「暗い」是反義詞，意思是昏暗。' },
        { text: '狭{せま}い', reason: '錯誤：「狭い」意思是狹窄，與明亮無關。' },
        { text: '静{しず}かだ', reason: '錯誤：「静か」意思是安靜，與明亮無關。' }
    ],
    correctIndex: 0,
    level: 'N5',
    tag: 'vocab_ruigigo',
    correctRule: '【考點】近義詞理解\n【解析】「明{あか}るい」表示有光、明亮，近義表達為「光{ひかり}がたくさんある」（有很多光）。\n【反義詞】暗{くら}い（昏暗的）'
}
```

#### 題型 6：用法（單字正確用法）

**考察重點**：單字在句中是否使用正確，包含語法搭配（Collocation）

**這是最難的題型**：不僅要懂單字意思，還要懂正確的語法搭配

**題目格式**：
```typescript
{
    id: 'n4_vocab_youhou_majime_1',
    prob: '「まじめ」の使{つか}い方{かた}が正{ただ}しいものはどれですか。',
    prob_zh: '下列哪個「認真」的用法是正確的？',
    options: [
        { text: '彼{かれ}はまじめな人{ひと}だ。', reason: '正確！「まじめ」是 na 形容詞，接名詞用「～な」。' },
        { text: '彼{かれ}はまじめい人{ひと}だ。', reason: '錯誤：「まじめ」不是 i 形容詞，不能加「い」。' },
        { text: '彼{かれ}はまじめの人{ひと}だ。', reason: '錯誤：na 形容詞接名詞用「～な」，不是「～の」。' },
        { text: '彼{かれ}はまじめく働{はたら}く。', reason: '錯誤：na 形容詞變副詞用「～に」，不是「～く」。' }
    ],
    correctIndex: 0,
    level: 'N4',
    tag: 'vocab_youhou',
    correctRule: '【考點】na 形容詞的用法\n【解析】「まじめ（真面目）」是 na 形容詞：\n・接名詞：まじめ＋な＋名詞（まじめな人）\n・變副詞：まじめ＋に（まじめに働く）\n【易錯點】\n・○ まじめな人\n・✕ まじめい人（誤判為 i 形容詞）\n・✕ まじめの人（助詞錯誤）'
}
```

## 6. 格式規範

### 6.1 prob（日文題目）

**單句格式**：
```
この部屋{へや}はとても<u>明るい</u>ですね。
```

**對話格式**（需換行）：
```
「この部屋{へや}は<u>明るい</u>ですね。」\n「はい、窓{まど}が大{おお}きいですから。」
```

**劃線標記**：使用 `<u>` 標籤標記需要選擇讀音或替換的單字

### 6.2 prob_zh（中文翻譯）

用 `*` 包裹答案：

**單句**：
```
這個房間非常 *明亮* 呢。
```

**對話**：
```
「這個房間很 *明亮* 呢。」\n「是的，因為窗戶很大。」
```

### 6.3 options.reason（選項理由）

| 類型 | 開頭 | 範例 |
|------|------|------|
| 正確 | `正確！` | `正確！「明るい」讀作「あかるい」。` |
| 錯誤 | `錯誤：` | `錯誤：多了長音，正確讀音沒有長音。` |

### 6.4 correctRule（正解詳解）

依題型不同，包含相應區塊：

**漢字読み / 表記**：
```
【考點】漢字讀音/書寫
【解析】說明正確讀音或漢字
【同音異義字比較】（如適用）
【易錯點】列出常見錯誤
```

**文脈規定**：
```
【考點】文脈語境判斷
【解析】說明為何選此詞
【相關詞比較】列出相似詞彙的差異
【文法補充】（如適用）
```

**語形成**：
```
【考點】接頭辭/接尾辭用法
【解析】說明詞素組合規則
【常見「～X」詞彙】列出同類詞彙
```

**類義語**：
```
【考點】近義詞理解
【解析】說明詞義相近的原因
【反義詞】（如適用）
```

**用法**：
```
【考點】詞類/語法搭配
【解析】說明正確用法
【易錯點】列出常見搭配錯誤
```

## 7. 選項設計原則

### 7.1 漢字読み 選項

| 類型 | 說明 |
|------|------|
| 正確答案 | 正確的平假名讀音 |
| 長音錯誤 | 多加或少加長音（あかるうい） |
| 清濁音錯誤 | 混淆清音濁音（あがるい） |
| 促音錯誤 | 多加或少加促音（あっかるい） |

### 7.2 表記 選項

| 類型 | 說明 |
|------|------|
| 正確答案 | 正確的漢字 |
| 同音異義字 | 讀音相同但意思不同的漢字 |
| 形似字 | 字形相似但不同的漢字 |
| 不存在的組合 | 無此讀音的漢字組合 |

### 7.3 文脈規定 選項

| 類型 | 說明 |
|------|------|
| 正確答案 | 符合語境的單字 |
| 反義詞 | 意思相反的詞 |
| 語意不合 | 文法對但語意不通的詞 |
| 詞性錯誤 | 詞性不符的詞 |

## 9. 工作流程

1. **選擇單字**：從 `src/data/vocabulary.ts` 選擇要出題的單字
2. **確認級別**：根據單字的 level 決定要出幾種題型
3. **逐題撰寫**：每種題型撰寫 2 題（單句 + 對話）
4. **檢查格式**：確認所有格式符合規範
5. **執行驗證**：`npx tsc --noEmit`
6. **加入檔案**：將題目加入對應的級別區塊

---

## 10. 注意事項

1. **漢字標註**：所有日文漢字都要加 `{假名}` 標註
2. **劃線標記**：漢字読み、表記、類義語題型需用 `<u>` 標記目標單字
3. **對話換行**：對話題目的日文和中文都要用 `\n` 換行
4. **錯誤選項**：必須有明確的錯誤原因，不能是「也可以但不是最佳」
5. **級別限制**：
   - N5/N4：不出「語形成」題型
   - N3：可出「用法」題型
   - N2/N1：可出全部 6 種題型
