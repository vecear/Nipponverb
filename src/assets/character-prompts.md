## 使用建議

1. **圖片尺寸**: 建議生成 512x512 或更高解析度，再縮小到 64x64 或 128x128 使用
2. **背景**: 統一使用 `#1e293b`（Tailwind slate-800）作為透明背景參考
3. **檔案格式**: PNG 含透明背景
4. **命名規範**: `{job_id}_{gender}_{stage_index}.png`
   - 例如: `doshin_male_01.png`, `hokan_female_10.png`

## 推薦工具

- **Midjourney**: 加上 `--niji 5` 參數獲得更好的動漫風格
- **DALL-E 3**: 直接使用 prompt
- **Stable Diffusion**: 使用 pixel art 專用模型如 `pixelart-v1`
