## 6. 職業階段變化

> **存放位置：** `src/assets/characters/stages/`
> **尺寸：** 512x768px (PNG with transparency)
> **命名規則：** `{jobId}_{stage}_{gender}.png`
> **階段範圍：** stage1 (Lv.5-14) 到 stage10 (Lv.95-99)

### 6.1 階段進化通用描述

在每個職業的基礎 prompt 後加入以下階段變化描述：

**Stage 1 (Lv.5-14) - 見習**
```
Add to base prompt: apprentice level, simple and modest attire,
humble expression, basic equipment, learning posture
```

**Stage 2 (Lv.15-24) - 初級**
```
Add to base prompt: beginner level, slightly better attire,
gaining confidence, basic professional tools
```

**Stage 3 (Lv.25-34) - 中級**
```
Add to base prompt: intermediate level, standard professional attire,
competent expression, proper equipment
```

**Stage 4 (Lv.35-44) - 上級**
```
Add to base prompt: advanced level, quality attire with details,
skilled expression, refined equipment
```

**Stage 5 (Lv.45-54) - 熟練**
```
Add to base prompt: expert level, distinguished attire,
confident master expression, excellent equipment
```

**Stage 6 (Lv.55-64) - 達人**
```
Add to base prompt: master level, prestigious attire with status symbols,
commanding presence, superior equipment
```

**Stage 7 (Lv.65-74) - 名人**
```
Add to base prompt: renowned level, famous figure attire,
authoritative expression, legendary equipment
```

**Stage 8 (Lv.75-84) - 宗師**
```
Add to base prompt: grandmaster level, ceremonial attire,
wise and powerful aura, mythical equipment
```

**Stage 9 (Lv.85-94) - 傳說**
```
Add to base prompt: legendary level, iconic attire,
ethereal presence, divine equipment, glowing accents
```

**Stage 10 (Lv.95-99) - 神話**
```
Add to base prompt: mythical level, transcendent attire with celestial elements,
god-like presence, legendary artifacts, golden aura
```

## 使用說明

1. **圖片生成工具推薦：** Midjourney, DALL-E 3, Stable Diffusion (with Japanese art LoRA)

2. **品質參數建議：**
   - Midjourney: `--v 6 --ar 2:3 --style raw` (角色), `--ar 1:1` (圖示)
   - DALL-E: 選擇 "vivid" style

3. **後處理建議：**
   - 使用 remove.bg 或 Photoshop 移除背景
   - 調整色彩以符合指定色碼
   - 統一輸出為 PNG 格式保持透明度

4. **檔案組織：**
   ```
   src/assets/
   ├── brand/
   │   └── logo.svg
   ├── textures/
   │   ├── washi-texture.png
   │   └── wave-pattern.png
   ├── icons/

   │   │   ├── course-n4.png
   │   │   └── ...
   │   └── practice/
   │       ├── practice-gojuon.png
   │       ├── practice-verbs.png
   │       └── ...
   ├── characters/
   │   ├── novice_male.png
   │   ├── novice_female.png
   │   ├── jobs/
   │   │   ├── doshin_male.png
   │   │   ├── doshin_female.png
   │   │   └── ...

   │       ├── doshin_stage1_male.png
   │       └── ...
   ├── ui/
   │   ├── hanko-stamp.png
   ```
