## 🎯 如何使用

### 在本地測試

1. **啟動開發伺服器**：
   ```bash
   npm run dev
   ```

2. **開啟瀏覽器**：
   前往 `http://localhost:5173`

3. **測試語言切換**：
   - 在登入頁面右上角點擊語言切換器
   - 選擇不同的語言
   - 觀察介面立即切換語言

### 在正式環境

部署後，使用者可以：
1. 訪問您的網站
2. 點擊語言切換器
3. 選擇他們偏好的語言
4. 整個應用程式即時切換

## 🔧 技術細節

### 安裝的套件
```json
{
  "i18next": "^23.x.x",
  "react-i18next": "^14.x.x",
  "i18next-browser-languagedetector": "^7.x.x"
}
```

### 核心功能
1. **i18next**：核心國際化框架
2. **react-i18next**：React 綁定，提供 `useTranslation` hook
3. **語言檢測器**：自動檢測瀏覽器語言

### 程式碼範例

**在元件中使用翻譯**：
```typescript
import { useTranslation } from 'react-i18next'

function MyComponent() {
  const { t } = useTranslation()

  return <h1>{t('dashboard.welcome')}</h1>
}
```

**帶參數的翻譯**：
```typescript
<h1>{t('dashboard.welcome', { name: 'John' })}</h1>
// 輸出: "Welcome back, John"
```
